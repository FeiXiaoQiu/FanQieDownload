把 fanqie-dl.feixiaoqiu.top 绑到 Vercel（国内比 *.vercel.app 好打开）

前提：Vercel 上已有项目 fan-qie-download，且已连 GitHub 仓库。

一、Vercel 添加域名
1. 打开 https://vercel.com/dashboard
2. 点进项目 fan-qie-download
3. 上方 Settings → Domains
4. 输入：fanqie-dl.feixiaoqiu.top → Add
5. 页面会显示要你改的 DNS（常见两种）：
   - CNAME 名称 fanqie-dl，目标 cname.vercel-dns.com
   - 或 A 记录指向它给的 IP（以页面显示为准）

二、改你的域名 DNS
1. 登录买域名的地方（阿里云 / 腾讯云 / Cloudflare 等）
2. 找到 feixiaoqiu.top 的解析
3. 把原来指向 GitHub Pages 的 fanqie-dl 记录删掉或改掉
4. 按 Vercel 显示的新建 CNAME（或 A 记录）
5. 保存，等 1～30 分钟

三、验证
1. 浏览器打开 https://fanqie-dl.feixiaoqiu.top/
2. 状态栏应类似「在线加速」
3. 搜索「抽象职校生存手册」应几秒出结果

四、可选
- GitHub 仓库 Settings → Pages 可先关掉，避免两套冲突
- 证书由 Vercel 自动签发（比过期的 Pages 证书好）

打不开 vercel.app 没关系，绑好自己的域名后用自己的域名访问即可。
