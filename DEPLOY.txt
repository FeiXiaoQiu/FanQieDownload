番茄小说下载器 — 部署说明

一、GitHub Pages（静态，已自动 Actions）
  推 main 后自动部署。只含静态文件，搜索较慢。

二、Vercel（推荐，搜索/封面加速）— 若 Deploy 失败按下面做
  1. https://vercel.com/new 导入 FeiXiaoQiu/FanQieDownload
  2. Framework Preset 选 Other（不要选 Python / Next）
  3. 若 Settings 里 Output Directory 写了 public：可保留
     仓库 vercel.json 已配置 build 输出到 public
  4. Install / Build 不要手动 Override 成乱命令（以 vercel.json 为准）
  5. Root Directory 留空 → Deploy

  失败对照：
  - No Output Directory named "public"
    → 已修：build 会生成 public/；Redeploy 即可
  - Framework 选成 Python
    → 改 Other，关掉错误 Override 再 Redeploy

  仓库已含：api/health.js api/search.js api/proxy.js + vercel.json
  本机完整下载仍用：node server.js 或 一键启动.bat

三、本机
  node server.js
