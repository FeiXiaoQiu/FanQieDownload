办法 1（推荐）：电脑本机运行

  1. 打开 GitHub 仓库 → 绿色 Code → Download ZIP，解压
  2. 先装 Node.js LTS （下一步下一步安装即可）
  3. 双击 一键启动.bat（Windows）或 一键启动.command（Mac）
  4. 浏览器打开 http://127.0.0.1:8787，搜索会很快

办法 2：装浏览器扩展（手机浏览器一般不行）

  1. Chrome / Edge 扩展商店搜 Allow CORS 或 CORS Unblock 并启用
  2. 回到本页，点下面「测试直连」；成功后直接搜索即可
